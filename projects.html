Data Engineering – Property Price Prediction

The report provides a detailed analysis of property prices in Malaysia, leveraging data engineering techniques to study the factors influencing real estate values across various regions. It utilizes web scraping and data crawling to build a dataset of over 2,184 property records, processed through Apache Spark and Hadoop ecosystems for efficient data storage and analysis. The data is warehoused in Apache Hive, split into training and testing datasets stored in Neo4j and MongoDB. Streaming capabilities are implemented with Apache Kafka and Spark Structured Streaming. Machine learning models, including decision trees and neural networks, are used to predict property prices, providing insights that benefit stakeholders like buyers, developers, and policymakers in making informed decisions.
