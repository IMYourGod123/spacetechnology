EHDEN Academy : ATLAS

This course, based on the OMOP CDM and Standardized Vocabularies workshop by OHDSI and extended for EHDEN Academy, provides an introduction to the history of OMOP and OHDSI’s vision for advancing observational research through network studies. It covers the structure and application of OMOP Standardized Vocabularies before delving into the Common Data Model (CDM), including key concepts like era logic. The course includes hands-on exercises, leveraging OHDSI-in-a-Box for practical learning. It also explores CDM version control, highlighting updates from v5.3 to v5.4, and features insights from Data Partners on real-world ETL use cases.

For more details, visit the EHDEN Academy : ATLAS

EHDEN Academy : Infrastructure

This course provides an in-depth guide to setting up and using OHDSI infrastructure and tools for working with patient-level data in the OMOP Common Data Model (CDM). It covers the installation and configuration of OHDSI’s WebAPI, a Java-based API for standardized analytics, and Atlas, a web application for designing and executing observational analyses. The course also includes the setup of R and RStudio, along with OHDSI R libraries for querying the OMOP CDM. Hands-on exercises use a provided virtual machine, with step-by-step video tutorials for installation on a Windows Server environment.

For more details, visit the EHDEN Academy : Infrastructure

EHDEN Academy : Extract, Transform and Load

This course covers the Extract, Transform, and Load (ETL) process for converting raw observational data into the OMOP Common Data Model (CDM). It explores tools and methods for designing ETL logic, including mapping source vocabulary values to Standard OMOP Concepts and handling unmapped data. The course also provides practical guidelines and examples for implementing and testing ETL logic to ensure accurate data transformation.

For more details, visit the EHDEN Academy : Extract, Transform and Load

EHDEN Academy : Usagi & Code Mappings for an ETL

This course provides hands-on training in using Usagi for mapping source system codes to standard terminologies in the OMOP Common Data Model (CDM). Participants will learn to import, review, and export code mappings for integration into an ETL pipeline. The course also covers advanced Usagi features such as filtering, auto concept ID assignment, concept information, and mapping types to streamline the mapping process. Prerequisites include foundational knowledge of OMOP CDM, Standardized Vocabularies, and ETL processes.

For more details, visit the EHDEN Academy : Usagi & Code Mappings for an ETL

EHDEN Academy : OHDSI-in-a-Box

This course teaches how to quickly deploy OHDSI-in-a-Box, a single-instance implementation of OHDSI tools with sample data for personal learning and training. It includes SynPUF data in OMOP CDM, a relational database with a query client, WebAPI, Atlas, R, and Python. The course also provides an optional introduction to OHDSI on AWS, a scalable, multi-user, and fault-tolerant OHDSI environment for enterprise use.

For more details, visit the EHDEN Academy : OHDSI-in-a-Box

EHDEN Academy : Phenotype Definition, Characterisation and Evaluation

This module provides in-depth training on defining, characterizing, and evaluating phenotypes using OHDSI tools such as Cohort Diagnostics, PheValuator, and ATLAS. Participants will learn a stepwise approach to using R tools for cohort analysis, identifying attributes and potential flaws, and assessing phenotype algorithms with PheValuator. The course also covers cohort design in ATLAS, including concept sets, grouping, and creation. By the end, learners will be proficient in building and critically evaluating cohorts. Prerequisites: ATLAS course and The Book of OHDSI chapter.

For more details, visit the EHDEN Academy : Phenotype Definition, Characterisation and Evaluation

EHDEN Academy : Patient-Level Prediction

This module provides a deep dive into patient-level prediction using OHDSI tools, particularly ATLAS. Led by Jenna Reps, developer of the OHDSI Patient-Level Prediction framework, the course covers best practices and key decision-making steps in the standardized prediction process used by OHDSI and EHDEN. Participants will learn how to develop, evaluate, and present prediction models while also focusing on defining relevant large-scale prediction questions.

For more details, visit the EHDEN Academy : Patient-Level Prediction

EHDEN Academy : R for Patient-Level Prediction

This module, led by Jenna Reps, provides hands-on training in using R for Patient-Level Prediction (PLP) within the OHDSI framework. Participants will explore the OHDSI R PLP library, learn to develop prediction models, and create and edit Prediction Study Packages using ATLAS. The course also covers building R Packages for external model validation, contributing study packages to the OHDSI-Studies GitHub repository, and enhancing the PLP Package by adding new classifiers or features.

For more details, visit the EHDEN Academy : R for Patient-Level Prediction

EHDEN Academy : Value Based Healthcare

This course, developed by the International Consortium for Health Outcomes Measurement (ICHOM), explores the principles of Value-Based Healthcare (VBHC) and its role in addressing rising healthcare costs, improving patient outcomes, and reducing disparities. Participants will learn about standardized, science-based measurement of patient-centered outcomes and how these data can be used for benchmarking and performance-based care. The course includes pre-recorded presentations and a self-assessment questionnaire, equipping learners with the knowledge to define patient health outcomes and apply standardized measurement practices in healthcare.

For more details, visit the EHDEN Academy : Value Based Healthcare

EHDEN Academy : Health Technology Assessment

This module, presented by NICE (National Institute for Health and Care Excellence), introduces the fundamentals of Health Technology Assessment (HTA)—the systematic evaluation of healthcare technologies for clinical and cost-effectiveness. Participants will explore key HTA concepts, common assessment approaches, and the role of real-world evidence (RWE) in informing decisions. The course includes a case study on chronic obstructive pulmonary disorder (COPD), showcasing how EHDEN was used in HTA processes. By the end, learners will understand the purpose of HTA agencies, their methodologies, and how RWE enhances healthcare decision-making. Prerequisites: EHDEN Foundation and Phenotype Definition, Characterisation, and Evaluation courses.

For more details, visit the EHDEN Academy : Health Technology Assessment
